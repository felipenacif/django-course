## Passo a passo

:heavy_check_mark: Instalar virtual env
```
$ python3 -m venv env
```   

:heavy_check_mark: Executar env
```
$ source env/bin/activate
```  

:heavy_check_mark: Instalar Django 
```
$ pip3 install django
```  

:heavy_check_mark: Rodar o server
```
$ python3 manage.py runserver
```
